3. Java란 무엇인가?
(1) 3대 프로그램 언어 중 하나
3대 : C, Java, Python, (Javascript)
    C : 속도, 하드웨어, cf)Assembly
    Java : 배포, 코드 재사용성, 웹
    Python : 쉬움, 다른 사람의 소스코드를 참조하기 편함
    
(2) 1991년 제임스 고슬링이 개발
    최초 이름 : OAK -> Java
    개발 목적 : 가전제품용 소형 OS

(3) 웹 애플리케이션의 발전에 따라 널리 퍼지게 됨

(4) 같은 코드로 어느 시스템이든 사용 가능
    Code Once, Runm Everywhere
    -JVM(Java Virtual Machine)이 설치된 곳이라 어디에서나

(5) 대표적인 객체지향 언어로 웹 프로그램의 대명사

[Compile  방식 vs Interpreter 방식]
Compile : 기계어로 전부 변환 > 실행 ex) C, Java
Interpreter : 매 줄(기본 단위) 기계어로 전환 > 실행
            오류나기 전까지는 실행됨
            ex) Python, Javascript

4. 주석(comment)
(1) 소스코드에 설명을 써 넣는 것
(2) 컴파일 시에 사용 안함
(3) 주석을 만드는 목적
    - 변수, 함수, 로직에 설명을 적기 위해
    -프로그램 로직이 복잡하여 주석으로 플로우(흐름)을 미리 기술
    조금씩 프로그램으로 전환
    (Pseudo Coding :  수도 코딩)
    -프로그램 시작 부분에 개요를 쓸 때
    -이미 수행 테스트를 한 부분을 건너 뛰고 싶을 때
(4) 한 줄 주석, 구간 주석, 문서화 주석
    한 줄 주석 : //
    구간 주석 : /* */
    문서화 주석 : /** */
