Day1

1. Node.js
    - "Node.js는 Chrome V8 javascript 엔진으로 빌드된
       Javascript 런타임"
      from Node.js 공식 홈페이지
    - Javascript의 해석은 브라우저가 담당하며
      브라우저 내에 있는 Javascript 엔진이 그 역할을 담당
    - 브라우저의 종류에 따라 사용하는 엔진이 다르며
      인터넷 브라우저에서만 Javascript 실행이 가능
    - Javascript V8 엔진으로 구글이 크롬을 개발,
      V8 엔진 성능이 너무 좋아서 새로운 프로젝트 시작
      -> Node.js(V8)이 만들어짐 -> 브라우저 외 환경에서 js 수행 가능
      -> 이게 뭐? -> Javascript가 웹 전용이 아니며 일반 프로그램 언어로 사용 가능해짐
    - Node.js가 웹 서버냐? 절대 아님
      아파치 웹 서버처럼 웹 디렉토리 설정, 서버 스타트 등 설정 불가
      -> 대신 http서버를 라이브러리를 통해 직접 작성/사용 가능
    - Node.js는 자바스크립트를 실행할 수 있는 하나의 환경
